# 快速开始指南

## 第一次使用

### 步骤1：安装依赖（只需执行一次）

```bash
# 方式1：使用自动安装脚本（推荐）
./install_dependencies.sh

# 方式2：手动安装
pip3 install -r requirements.txt
```

### 步骤2：运行爬虫

```bash
# 推荐使用改进版（PDF质量更好）
python3 scrape_scys_advanced.py
```

### 步骤3：首次登录

当浏览器窗口打开后：

1. 在浏览器中输入你的账号密码登录
2. 确认登录成功后
3. 返回终端窗口
4. 按下回车键继续

✅ 登录信息会自动保存，下次运行无需再次登录！

### 步骤4：等待完成

脚本会自动：
- 查找热门文章（前5篇）
- 逐个访问并保存为PDF
- 显示进度和结果

所有PDF文件将保存在 `scys_pdfs/` 目录中。

## 常见问题

### Q: 如何重新登录？

删除cookies文件后重新运行：
```bash
rm scys_cookies.json
python3 scrape_scys_advanced.py
```

### Q: 找不到热门文章怎么办？

1. 检查是否正确登录
2. 查看生成的 `debug_page.html` 文件
3. 可能需要根据网站最新结构调整代码

### Q: 如何更改保存的文章数量？

编辑脚本，找到这一行：
```python
return articles[:5]  # 改为你想要的数量
```

### Q: 可以定期自动运行吗？

可以使用cron定时任务：
```bash
# 编辑crontab
crontab -e

# 添加一行（每天早上9点运行）
0 9 * * * cd /path/to/project && python3 scrape_scys_advanced.py
```

## 两个版本的区别

### 改进版（推荐）：`scrape_scys_advanced.py`
- ✅ 使用浏览器原生打印功能
- ✅ PDF保留完整网页样式
- ✅ 保留图片和格式
- ✅ 无需安装字体
- ⚠️ 需要Chrome浏览器

### 基础版：`scrape_scys.py`
- ✅ 使用Python库生成PDF
- ❌ 纯文本，无样式
- ❌ 不包含图片
- ⚠️ 需要安装中文字体
- ✅ 更轻量级

## 技巧和建议

### 1. 批量下载时避免被封
```python
# 在脚本中增加等待时间（已默认设置为2秒）
time.sleep(2)  # 修改为更长时间，如 5 秒
```

### 2. 保存到指定目录
```python
# 修改输出目录
self.output_dir = "/path/to/your/pdfs"
```

### 3. 自定义PDF设置
```python
# 在 save_page_as_pdf 方法中修改
pdf_settings = {
    'landscape': False,      # 改为 True 可横向打印
    'paperWidth': 8.27,      # A4纸宽度
    'paperHeight': 11.69,    # A4纸高度
    'marginTop': 0.4,        # 上边距
    # ... 可以调整更多参数
}
```
